Politique de confidentialité

Dernière mise à jour : 22 février 2026

Présentation

Song Chords est une application web personnelle permettant d'afficher et de formater des partitions d'accords de chansons. Cette politique décrit comment vos données sont traitées.

Données collectées

Song Chords ne collecte aucune donnée personnelle sur un serveur. Toutes les données sont stockées localement dans votre navigateur (localStorage) :

  • Vos chansons et leurs paramètres d'affichage
  • Vos préférences d'interface (taille de police, couleurs, colonnes)
  • Vos identifiants Ultimate Guitar (si vous utilisez la recherche web)

Google Drive

Si vous choisissez de connecter Google Drive, l'application accède à votre compte Google uniquement pour synchroniser vos chansons dans un dossier dédié. L'application :

  • Ne lit que les fichiers qu'elle a elle-même créés dans le dossier Song Chords
  • Ne partage vos données avec aucun tiers
  • Ne stocke pas votre token Google sur un serveur (il reste dans votre navigateur)

Ultimate Guitar

Si vous utilisez la fonctionnalité de recherche web, vos identifiants Ultimate Guitar sont :

  • Stockés uniquement dans le localStorage de votre navigateur
  • Transmis de manière chiffrée (HTTPS) à notre proxy serveur pour authentification auprès de l'API Ultimate Guitar
  • Jamais stockés ni journalisés côté serveur

Cookies et tracking

Song Chords n'utilise aucun cookie, aucun outil d'analyse, aucun tracker.

Suppression des données

Vous pouvez supprimer toutes vos données à tout moment en vidant le localStorage de votre navigateur (Paramètres du navigateur → Données de site).

Contact

Pour toute question, vous pouvez ouvrir une issue sur le dépôt du projet.
